# श्रीमद्भगवद्गीता • Bhagavad Gita Reader

An open-source, ad-free, and fully offline **Bhagavad Gita Reader for Android** designed for immersive reading, contemplation, and daily spiritual practice.

Whether you search for **Bhagavad Gita**, **Bhagvad Gita**, **Bhagwad Gita**, **Bhagwat Geeta**, **Bhagavad Geeta**, or simply **Geeta**, this app provides a clean and distraction-free reading experience with Sanskrit shlokas, Hindi translations, English translations, word meanings, bookmarks, notes, and daily wisdom—all without requiring an internet connection.

Built with a modern hybrid architecture (Kotlin + Web technologies), the app combines elegant design, responsive Devanagari typography, multilingual support, and offline-first performance to create a premium Bhagavad Gita reading experience.

---

## ✨ Features

### 📖 Complete Bhagavad Gita Reading Experience

* Read all chapters and verses of the **Shrimad Bhagavad Gita** in their original Sanskrit form.
* Access Hindi and English translations alongside each verse.
* Explore detailed word meanings for deeper understanding and study.
* Optimized Devanagari typography using Google's **Mukta** font for comfortable long-form reading.

### 🎨 Clean, Modern & Distraction-Free Design

* Minimalist reading interface focused entirely on scripture.
* Elegant glassmorphism-inspired UI with smooth animations and responsive layouts.
* Warm saffron-accented design inspired by traditional Indian aesthetics.
* No advertisements, popups, or unnecessary distractions.

### ⚙️ Personalized Reading Controls

* Adjustable font size for comfortable reading.
* Toggle Sanskrit, Hindi, and English independently.
* Intelligent validation ensures at least one translation remains visible.

### 🔖 Bookmarks & Spiritual Journal

* Save favorite verses for quick access.
* Maintain a personal diary for reflections, interpretations, and daily learnings.
* Build your own collection of meaningful Bhagavad Gita teachings.

### 🕉 Daily Wisdom Widget

* Display a Bhagavad Gita quote directly on your Android home screen.
* Automatic daily verse updates.
* Beautiful frosted-glass widget design.

### 🌅 Meditation & Quiet Hours

* Schedule daily reminders for reading, meditation, and reflection.
* Fully offline notifications with no external services required.

### 🔒 Private & Offline First

* 100% offline Bhagavad Gita reader.
* No trackers.
* No analytics.
* No ads.
* No account required.
* All bookmarks, notes, and settings remain on your device.

## 🚀 Installation & Build Guide

### Prerequisites

* JDK 17 or higher
* Android SDK (API Level 36 compile target)
* Android Studio (latest stable version recommended)

### Clone the Repository

```bash
git clone https://github.com/ayushsoni/bhagvadgitareader.git
cd bhagvadgitareader
```

### Sync Web Assets

Ensure the following files:

```text
index.html
index.css
app.js
gita_data.js
```

are available inside:

```text
android-app/app/src/main/assets/
```

### Build the APK

```bash
cd android-app
./gradlew assembleDebug
```

### Install on a Device or Emulator

```bash
adb install -r app/build/outputs/apk/debug/app-debug.apk
```
